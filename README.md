# 3D House Map Maker

A simple browser app for sketching house layouts on a 2D grid and previewing them as a 3D model.

## Features

- Adjustable map width and depth
- Adjustable wall height
- Click-to-toggle wall cells in a top-down grid
- One-click "New Plan" perimeter generator
- Real-time 3D preview with orbit controls

## Run locally

```bash
python3 -m http.server 4173
```

Open <http://localhost:4173> in a browser.
